# Angular Beginner's Toolkit with GenAI

## Project Overview
This beginner-friendly Angular toolkit was built using Generative AI prompts. It demonstrates a minimal working example (MWE) of Angular with interactive features.

## Features
- Displays a greeting message
- Button toggles message text
- Input field updates a personalized greeting
- Demonstrates component-based architecture and Angular binding

## Setup Instructions
1. Install Node.js (v18+) and npm
2. Install Angular CLI globally:
